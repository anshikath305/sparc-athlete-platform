# SPARC – Athlete Performance Platform

SPARC is a SwiftUI-based athlete performance and resilience tracking platform submitted to the Apple Swift Student Challenge.

## Features

- Athlete performance dashboard
- Workout logging
- Mental resilience reflections
- Achievement timeline
- AI performance insights
- Shareable athlete portfolio

## Built With

- Swift
- SwiftUI
- Swift Playgrounds (.swiftpm)
- Codable persistence
- Combine state management

## Purpose

SPARC aims to help athletes track their physical progress, mental resilience, and professional growth while building a shareable sports portfolio.
